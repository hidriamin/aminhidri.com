Simple Dark Mode implementation

I'm finally writing a blog! for a first entry, i wanted to talk about some work i did on my personal website. I'm not the most professional programmer in the work, but you might judge me being good at problem solving. I never added dark mode to a website before, this is my first. While working on my personal website (during the night time) i maybe thought it was too bright. I started implementing Dark Mode to it. I wrote a function that finds the Link tag that has the styles, and switches the Href to the light or dark mode styles.
const mode = () => { const linkTag = document.querySelector("link"); linkTag.href.endsWith("styles.css") ? (linkTag.href = "darkStyles.css") : (linkTag.href = "styles.css"); }; Yes, i made 2 CSS files: one for Dark mode and one for light mode:
